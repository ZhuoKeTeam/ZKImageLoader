# ZKImageLoader
ZKImageLoader

## 使用说明

在任何地方直接使用如下代码
```
ZKImageLoader.instance.load(this, imageView, url)
```

默认包含了占位图，错误图，只需要写 url 就可以。

this 可以是上下文 Context， 也可以是 Activity 及 相关类，再或是 Fragment，只用 support v4 的哦。
